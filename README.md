## 🛠️ Tech Stack & Architecture

* **Framework:** React 18 (Vite-powered for high-performance builds)
* **Language:** TypeScript (Type-safe component design and API validation)
* **Styling:** Tailwind CSS (Modern responsive design systems, transitions, and layout grids)
* **API Client:** Axios (Integrated with TanStack React Query for reliable state mutation)
* **Vector Icons:** Google Material Symbols

---

## 🚀 Key Features Implemented

1. **Cinematic Hero Video Background:** Homepage utilizes a full-bleed looping technology network video background. Overlaid with high-contrast, professional typography that remains perfectly readable.
2. **Premium Split & Row Banners:** Capabilities pages (Services, Products, About Us) feature full-width graphic rows with clean, slightly opaque overlays (`opacity-85`) that integrate text and branding.
3. **Vanilla Pathname Router:** Implemented a custom path-matching engine synchronized with HTML5 History API (`window.history.pushState` and `popstate` listeners) allowing clean URL paths (e.g. `/services`, `/about`) without full-page reloads.
4. **Integrated Footer Subscription:** Interactive form mapping to backend endpoints with automated admin notifications and receipt responders.

## 📦 Local Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/diva-sl/gcv-digital-engineering.git
   cd gcv-digital-engineering
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory:
   ```env
   VITE_API_URL=http://localhost:5000/api
   ```

4. **Run Development Server:**
   ```bash
   npm run dev
   ```

5. **Build for Production:**
   ```bash
   npm run build
   ```
